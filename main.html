What is MediVault?

MediVault is a digital health record management system that lets patients, doctors, and admins interact on a secure platform.

  • Patients → store & view their medical history.
  • Doctors → add diagnoses & prescriptions.
  • Admins → manage users and oversee the system.

It’s like a mini electronic health record (EHR) system that hospitals use, but built lightweight for quick deployment.

Why MediVault? (Problem Statement)
  • Patients often lose paper prescriptions or reports.
  • Doctors lack a centralized record system to track patient history.
  • Hospitals need a secure, role-based platform to manage data.

MediVault solves this by providing a web-based, easy-to-use, secure solution

Features (MVP version)
  1. User Authentication
    • Register/Login (Patient, Doctor, Admin)
    • Role-based dashboards
  2. Patient Dashboard
    • Patient Dashboard
  3. Doctor Dashboard
    • Add medical records for patients
    • Attach prescriptions
  4. Admin Dashboard
    • Manage patients & doctors
    • Oversee the system
Tech Stack
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Python (Flask/Django)
  • Database: MySQL
  • UI/UX: Responsive Bootstrap 5 layout
Challenges Solved
  • Ensuring data privacy (role-based access)
  • Making it hackathon-friendly (lightweight + quick to deploy)
  • Using a modular structure so extra features can be added later
